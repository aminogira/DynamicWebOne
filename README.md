# Multi Git Test Project

* **Project Name**: Multi Git Test Project
* **Purpose**: Testing multiple Git account configuration
* **Author**: Sample Developer

## Description
This is a small multi-file Java EE web application designed to demonstrate and verify Git configuration across multiple accounts (e.g., personal and work). It includes a simple greeting servlet, utility classes, and model objects to establish a compileable codebase.
